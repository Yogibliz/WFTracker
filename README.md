# Installation

- Clone the repo (```git clone git@github.com:Yogibliz/WFTracker.git```)

- Get virtualenv if not already installed (```pip install virtualenv```)
- Run ```virtualenv venv``` to create the virtual environment
- Source the virtual environment using ```source venv/bin/activate``` to enter it.
- Install requirements ```pip install -r requirements.txt```

- Run using python or setting executable permissions

It's easiest to run using: ```python3 WFTracker.py```
